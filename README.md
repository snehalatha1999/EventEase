# EventEase - Blazor Event Management App

EventEase is a web application for browsing and registering for corporate and social events. Built using **Blazor** and **ASP.NET Core**, this app leverages **Microsoft Copilot** for rapid development, debugging, and optimization.

---

## Features

- Browse upcoming events with details: name, date, location
- Event registration with **form validation** (Name & Email)
- **User session management** to track logged-in users and registered events
- **Attendance tracker** to monitor event participation
- Responsive UI with optimized rendering for large datasets
- Graceful handling of invalid routes and inputs

---

- [Blazor](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor)
